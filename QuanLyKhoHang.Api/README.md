# QuanLyKhoHang.Api

Backend API server rieng cho ung dung WinForms quan ly kho.

## Chay API

```powershell
dotnet run --project QuanLyKhoHang.Api\QuanLyKhoHang.Api.csproj
```

Mac dinh API lang nghe tai:

```text
http://localhost:5088
```

WinForms doc dia chi nay tu `Config\appsettings.json` trong phan `ApiClientSettings`.
